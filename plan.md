## siga esse passo corretamente

1. crie uma api java com spring boot
2. faça o commit e o push para o github 
3. crie uma nova branch chamada "feature/cadastro-usuario"
4. implemente a funcionalidade de cadastro de usuário
5. faça o commit e o push para a branch "feature/cadastro-usuario"
6. crie um pull request para a branch "developer" e aguarde a revisão do código
7. após a aprovação do pull request, faça o merge da branch "feature/cadastro-usuario" para a branch "developer"
8. faça o commit e o push para a branch "developer"
